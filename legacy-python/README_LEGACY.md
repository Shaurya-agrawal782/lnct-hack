# Legacy Python PyQt5 Prototype

This folder contains the original Python/PyQt5 desktop prototype for DisasterConnect.

## Overview
- **Purpose**: It is preserved here as a reference for the MERN migration.
- **Main Reusable Concepts**:
  - Authentication (auth)
  - Incident tracking (incidents)
  - Resource allocation (resources)
  - Interactive map integration (maps)
  - Reporting and data visualization (reports)
- **UI Code**: The PyQt5 UI code will not be directly reused in the new MERN web application.
