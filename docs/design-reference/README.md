# Stitch UI Design Reference

This directory contains visual design and layout references for the DisasterConnect UI redesign.

## Important Guidelines

1. **Visual Reference Only**
   - The HTML, CSS, and asset files in the `stitch-full-ui` directory are static layout and styling guides.
   - Do **NOT** copy or replace the existing React components directly with these standalone HTML files.

2. **Preserve Application Logic**
   - Any UI redesign must preserve the existing application state, core React logic, router configurations, authentication context, APIs, Socket.io integration, Map/Leaflet functions, and role-based access protections.

3. **Controlled Integration**
   - Implementation should be done incrementally through controlled React component refactoring and modern styling integration, matching the layout and aesthetics defined in these reference templates.
